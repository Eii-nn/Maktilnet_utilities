Desktop one-click launcher:

1. Double-click launch_firmwaredowngrade.bat to start the tool.
2. If Windows asks which app should open .bat files, choose Command Prompt or Windows Terminal.
3. If Python is not found, install Python from https://www.python.org/downloads/ and check "Add Python to PATH" during setup.
4. Install requirements once:
   py -3 -m pip install -r requirements.txt
